# Apple Catalog Store

Apple Catalog Store is a responsive e-commerce web application for browsing Apple devices and accessories, viewing product details, managing favourites, and adding items to the cart. The project focuses on clean product presentation, category-based navigation, and a smooth shopping flow across desktop, tablet, and mobile screens.

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/zaurmamedov/Apple__Catalog-Store.git
cd Apple__Catalog-Store
```

2. Install dependencies:

```bash
npm install
```

3. Run the project locally:

```bash
npm start
```

4. Build the project for production:

```bash
npm run build
```

## Features

- Product catalog with category-based navigation
- Product details page with gallery, colors, capacities, and technical specs
- Favourites management
- Shopping cart with quantity controls and total price calculation
- Responsive layout for mobile, tablet, and desktop
- Sliders for brand-new models and hot prices

## Key Challenges

- Building a scalable catalog structure with reusable product cards and shared page layouts
- Synchronizing favourites and cart state across different pages
- Implementing dynamic product details with color and capacity switching
- Creating a responsive UI that stays consistent across multiple breakpoints
- Organizing SCSS architecture and reusable typography/layout utilities

## Project Structure Notes

- `src/components` contains reusable UI building blocks
- `src/pages` contains route-level pages
- `src/styles` and `src/utils` contain shared styles, helpers, and assets mapping
